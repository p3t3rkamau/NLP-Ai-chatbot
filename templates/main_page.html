About The Code

Flask is a Python web framework that allows building web applications easily. render_template is a function that is used to render HTML templates in Flask. request is a Flask module that provides the request object, which contains information about the incoming HTTP request. jsonify is a Flask module that helps create JSON responses. Create a Flask application: app = Flask(__name__) creates a new Flask application. Define the API key: valid_api_keys = 'api_key_1' is the API key that is used to authenticate requests. Define a route for the homepage: @app.route('/') is a decorator that associates the function below it with the URL of the root directory. def index(): is the function that will be executed when the root directory is accessed. return render_template('index.html') renders the index.html file as the response to the client. Define a route for the chatbot: @app.route('/chatbot', methods=['POST']) associates the function below it with the URL /chatbot and allows only POST requests. def chatbot(): is the function that will be executed when the /chatbot URL is accessed. user_message = request.form['user_message'] retrieves the user's message from the form data submitted via the POST request. payload = {"user_message": user_message} creates a JSON object with the user's message. response = requests.post(url, headers={'api_key': valid_api_keys}, json=payload) sends a POST request to the specified API endpoint URL with the user's message and the API key as headers. chatbot_response = response.json()["chatbot_response"] extracts the chatbot's response from the JSON response. return render_template('index.html', chatbot_response=chatbot_response) renders the index.html file with the chatbot's response as a parameter. Start the Flask application: if __name__ == '__main__': checks if the script is being run directly. app.run(debug=True, port=5001) starts the Flask application on port 5001 with debug mode enabled.
