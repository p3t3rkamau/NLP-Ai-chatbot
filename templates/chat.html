from flask import Flask, render_template, request, jsonify
import requests

app = Flask(__name__)

# Define the API key
valid_api_keys= 'api_key_1'

@app.route('/')
def index():
    return render_template('index.html')

@app.route('/chatbot', methods=['POST'])
def chatbot():
    # Check if API key is valid
    # Set the API endpoint URL
    url = "http://localhost:5000/api/chat"

    # Set the user input message from the form data
    user_message = request.form['user_message']

    # Set the payload as a JSON object
    payload = {"user_message": user_message}

    # Send the POST request and get the response
    response = requests.post(url, headers={'api_key': valid_api_keys}, json=payload)

    # Get the chatbot's response from the response JSON object
    chatbot_response = response.json()["chatbot_response"]

    # Render the chatbot response on the page
    return render_template('index.html', chatbot_response=chatbot_response)

if __name__ == '__main__':
    app.run(debug=True, port=5001)

                                    
                
              
            
                 
About The Code

    Flask is a Python web framework that allows building web applications easily.
    render_template is a function that is used to render HTML templates in Flask.
    request is a Flask module that provides the request object, which contains information about the incoming HTTP request.
    jsonify is a Flask module that helps create JSON responses.
    Create a Flask application:

    app = Flask(__name__) creates a new Flask application.
    Define the API key:

    valid_api_keys = 'api_key_1' is the API key that is used to authenticate requests.
    Define a route for the homepage:

    @app.route('/') is a decorator that associates the function below it with the URL of the root directory.
    def index(): is the function that will be executed when the root directory is accessed.
    return render_template('index.html') renders the index.html file as the response to the client.
    Define a route for the chatbot:

    @app.route('/chatbot', methods=['POST']) associates the function below it with the URL /chatbot and allows only POST requests.
    def chatbot(): is the function that will be executed when the /chatbot URL is accessed.
    user_message = request.form['user_message'] retrieves the user's message from the form data submitted via the POST request.
    payload = {"user_message": user_message} creates a JSON object with the user's message.
    response = requests.post(url, headers={'api_key': valid_api_keys}, json=payload) sends a POST request to the specified API endpoint URL with the user's message and the API key as headers.
    chatbot_response = response.json()["chatbot_response"] extracts the chatbot's response from the JSON response.
    return render_template('index.html', chatbot_response=chatbot_response) renders the index.html file with the chatbot's response as a parameter.
    Start the Flask application:

    if __name__ == '__main__': checks if the script is being run directly.
    app.run(debug=True, port=5001) 
    starts the Flask application on port 5001 with debug mode enabled.
             
                 
How to run the code locally

    Install the necessary libraries:
    Flask: pip install flask
    Requests: pip install requests
    Obtain an API key:

    This code requires an API key to authenticate requests to the API endpoint.
    You can obtain an API key by contacting the owner of the API endpoint.
    Set the API endpoint URL:

    The url variable in the chatbot function specifies the API endpoint URL.
    Replace http://localhost:5000/api/chat with the actual URL of the API endpoint.
    Run the Flask application:

    Save the code in a Python file (e.g., app.py).
    Open a terminal and navigate to the directory where the Python file is saved.
    Run the Flask application with the command python app.py.
    The Flask application should start running on port 5001.
    Access the chatbot interface:

    Open a web browser and navigate to http://localhost:5001/.
    You should see a web page with a form that allows you to enter a message.
    Enter a message and submit the form.
    The chatbot should respond with a message.
